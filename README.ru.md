# ChatTabs Reloaded — Русский

Это форк мода ChatTabs, поддерживаемый Wakcedon (форк от Gri11edHam).

Описание
- Добавляет настраиваемые вкладки чата, фильтры сообщений и модификаторы отправки (префикс/суффикс).
- Поддержка профилей серверов — разные наборы вкладок для разных серверов.
- GUI конфигурации доступен в сборке для Fabric (Cloth Config). NeoForge-артефакт не зависит от Cloth.

Установкa
- Скопируйте соответствующий JAR из `fabric/build/libs` или `neoforge/build/libs` в папку `mods` вашей сборки Minecraft.

Поддерживаемые языки
- Английский (по умолчанию)
- Русский (включён)

Как работает мод
- Общий код (в `src/main/java`) реализует логику вкладок и конфигурацию в нейтральном виде.
- Fabric-мод содержит специфичные для Fabric входные точки и экран конфигурации.
- NeoForge-мод содержит обёртки/метаданные для NeoForge и минимальные зависимости.
